GitHub repositories

Linux Puppet modules:

  • autohome: configure automounted home directories
  • iptables: manage iptables incoming rules
  • limits: set entries in /etc/security/limits.conf
  • pamldap: configure LDAP host user authentication, authorisation and name resolution
  • ntp: manage the Network Time Server daemon (ntpd) on a Linux host
  • pdns: manage a PowerDNS namserver or resolver, with support for running internal domains
  • runit: manage Runit user services
  • ssh: manage SSH host and user access
  • system: manage Linux system resources and services from hiera configuration

Middleware Puppet modules:

  • jboss: install and run JBoss under Runit
  • sunjdk: install the Sun (Oracle) JDK from a tarball
  • tomcat: Install and run Apache Tomcat under Runit
  • weblogic: Install Weblogic (optionally with SOA and OSB)
  • wso2: Install and run WSO2 products (like ESB and Identity Server)

Other Puppet modules:

  • anthillpro: install and configure an Anthill Pro agent
  • logscape: install and configure a Logscape agent
  • ccfe: create Curses Command Front-end (CCFE) text console menus

Many of these have been released and are available on Puppet Forge.

Perl modules:

  • Tuxedo::Admin: Runtime administration of a Tuxedo environment
